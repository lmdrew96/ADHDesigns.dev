## The Chaos Ecosystem

### ChaosCode 🌪️ Brewing
A custom multi-LLM agentic IDE built on Electron with Monaco Editor, React, TypeScript, and Tailwind. Features a dual-LLM panel architecture — Claude Haiku handles implementation while Claude Sonnet reviews — designed for agentic development workflows.

### Cha(t)os 🔥 Raging
A multi-user group chat app where everyone brings their own Claude API key. Each user gets a personalized Claude instance in shared rooms with @mention routing, multimodal file uploads, and Personal Context MCP integration. Built on Convex + Next.js.
**Live:** Beta testing

### ControlledChaos 🔥 Raging
An ADHD-friendly task manager with a Crisis Mode feature for overwhelm moments. Includes a 10-tool MCP server for Claude Code integration.
**Live:** [controlledchaos.adhdesigns.dev](https://controlledchaos.adhdesigns.dev)

### ChaosLimbă 🔥 Raging
An English-to-Romanian language learning platform grounded in second language acquisition theory. Features a 10-component AI ensemble, a 3-tier fossilization Adaptation Engine, 14 dashboard routes, and a dedicated MCP server for instructional design auditing.
**Live:** [chaoslimba.adhdesigns.dev](https://chaoslimba.adhdesigns.dev)

### ChaosPatch ⚡ Unleashed
A lightweight dev patch tracker PWA with Claude Code MCP integration. Manages bugs, fixes, and refactors scoped to individual projects.
**Live:** [chaospatch.adhdesigns.dev](https://chaospatch.adhdesigns.dev)

## Other Projects

### ScribeCat 🌀 Sustained
A collaborative study platform with real-time speech-to-text transcription, multiplayer study games, and a Chrome extension. Built on Convex.

### ThreadBrain 🌀 Sustained
An ADHD-focused AI reading companion that helps break down and engage with dense text.

### ThreadNotes 🌀 Sustained
A research journal app for tracking academic research questions across thematic categories. Includes a browser extension for highlighting and saving excerpts from sources.
**Live:** [research.adhdesigns.dev](https://research.adhdesigns.dev)

### ChickenScratch 🔥 Raging
The editorial submission portal for Hen & Ink Society's literary zine, *Chicken Scratch*. Handles submissions, editorial review, and image processing.
**Live:** [chickenscratch.me](https://chickenscratch.me)

### Personal Context MCP 🌀 Sustained
An MCP server that stores identity, projects, relationships, and preferences for cross-session Claude context. Powers Cha(t)os context awareness and provides continuity across Claude conversations.
